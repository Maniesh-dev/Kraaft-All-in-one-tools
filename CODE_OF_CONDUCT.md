# Code of Conduct

## Our commitment

We want Kraaft to be an open, welcoming, and safe project for everyone.

Contributors, maintainers, and community members are expected to help create an environment that is respectful, inclusive, and harassment-free.

## Expected behavior

- Be respectful in communication
- Assume good intent, while giving honest and constructive feedback
- Be welcoming to people with different backgrounds and experience levels
- Focus on what is best for the project and the community
- Accept feedback gracefully

## Unacceptable behavior

- Harassment, intimidation, or discrimination
- Trolling, insulting, or derogatory comments
- Personal attacks
- Publishing private information without permission
- Any conduct that would be inappropriate in a professional setting

## Scope

This Code of Conduct applies to:

- GitHub issues and pull requests
- Discussions related to the project
- Community spaces linked to the project
- Public interactions where someone is representing the project

## Reporting

If you experience or witness unacceptable behavior, report it to the project maintainer, Maniesh Sanwal, through a private communication channel available on the repository hosting platform.

Project website: https://kraaft.manieshsanwal.in

Reports will be reviewed promptly and handled as respectfully and confidentially as possible.

## Enforcement

Project maintainers are responsible for clarifying and enforcing this Code of Conduct. They may take appropriate action in response to behavior they deem inappropriate, threatening, offensive, or harmful.

Possible actions include:

- Warning the contributor
- Removing or editing offending content
- Temporarily restricting participation
- Permanently banning a contributor from project spaces
